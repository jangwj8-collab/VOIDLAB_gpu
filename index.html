관리자 설정 가이드
Google Sheets를 데이터베이스로 연결하는 방법입니다.
1
Google Sheets 파일 만들기
Google Sheets에서 새 파일을 만들고, A1부터 아래 헤더를 입력하세요:
id   server   gpu   date   slot   name   purpose   created_at
2
Apps Script 코드 붙여넣기
Sheets 메뉴 → 확장 프로그램 → Apps Script → 아래 코드를 전체 복사해서 붙여넣고 저장하세요.
const SHEET_NAME = 'Sheet1';

function doGet(e) {
  const sheet = SpreadsheetApp
    .getActiveSpreadsheet().getSheetByName(SHEET_NAME);
  const rows = sheet.getDataRange().getValues();
  const headers = rows[0];
  const data = rows.slice(1).map(r => {
    const obj = {};
    headers.forEach((h, i) => obj[h] = r[i]);
    return obj;
  });
  return ContentService
    .createTextOutput(JSON.stringify(data))
    .setMimeType(ContentService.MimeType.JSON);
}

function doPost(e) {
  const sheet = SpreadsheetApp
    .getActiveSpreadsheet().getSheetByName(SHEET_NAME);
  const data = JSON.parse(e.postData.contents);
  if (data.action === 'delete') {
    const rows = sheet.getDataRange().getValues();
    for (let i = rows.length - 1; i >= 1; i--) {
      if (rows[i][0] == data.id) { sheet.deleteRow(i+1); break; }
    }
  } else {
    sheet.appendRow([data.id, data.server, data.gpu,
      data.date, data.slot, data.name,
      data.purpose, data.created_at]);
  }
  return ContentService
    .createTextOutput(JSON.stringify({ok:true}))
    .setMimeType(ContentService.MimeType.JSON);
}
3
웹앱으로 배포
Apps Script에서 배포 → 새 배포 → 웹앱 선택.
액세스 권한: "모든 사용자"로 설정 후 배포 → 생성된 URL 복사.
4
index.html 상단 SCRIPT_URL에 붙여넣기
이 파일 스크립트 최상단의 const SCRIPT_URL = ''; 부분에 복사한 URL을 붙여넣으세요.
5
서버·GPU 정보 수정
스크립트 상단 const SERVERS = [...] 배열을 실제 서버 이름과 GPU 개수에 맞게 수정하세요.
✅ 설정 완료 후: 팀원이 사이트에서 예약하면 Google Sheets에 자동으로 기록되고, 모든 사람의 달력이 동기화됩니다.
